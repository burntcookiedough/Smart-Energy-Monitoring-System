A browser-based smart home energy dashboard that simulates appliance loads, overload alerts, consumption analytics, and household energy behavior.

## What It Shows

- Live power consumption dashboard
- Appliance-level simulation controls
- Alert history for overload and anomaly events
- Consumption analytics with charted historical usage

## Project Shape

```text
index.html      # dashboard shell
app.js          # UI routing, charts, alert interactions
simulation.js   # appliance and energy simulation state
styles.css      # visual system and responsive layout
```

## Run Locally

Open `index.html` in a browser. No backend or build step is required.

## Why It Matters

This project frames energy monitoring as an interactive control surface: the user can simulate appliance behavior, observe load changes, and understand how overload conditions propagate through a smart home dashboard.
